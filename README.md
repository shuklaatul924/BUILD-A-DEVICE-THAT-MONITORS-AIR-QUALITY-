# BUILD-A-DEVICE-THAT-MONITORS-AIR-QUALITY-
# Air Quality Monitoring System using ESP8266
This project monitors air quality using MQ-135 sensor and sends data to blynk app.

### Components
-NodeMCU ESP8266
-MQ-135 Air Quality Sensor
-jumper wires 
-Breadboard
